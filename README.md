# Sistema de cadastro de clientes em um hotel

Sistema para cadastro de cliente, reserva de quarto, custo total de estadia e emissão de relatório final.
Desenvolvido no curso de Developer-TI da UNISUL.

### Como funciona
É solicitado o cadastro do cliente pedindo Nome e CPF.
Pergunta-se quantos hóspedes ficarão no quarto. É verificado se atende à quantidade máxima permitida.
É solicitado também a quantidade de Diárias, caso seja maior que 7 é aplicado um desconto ao valor total.
Ao final é emitido um relatório como a seguir:

DADOS DA ESTADIA
--------------------------
Informações do hóspede
Nome: FERNANDO REBELO
CPF: 123.123.123-12
--------------------------
Informações do quarto
Quarto número: 5
Quantidade máxima de hóspedes: 4
--------------------------
Detalhes da hospedagem
Quantidade de hóspedes: 4
Diárias: 8
Valor da diária: 200.0
Valor total a pagar: 1440.0
